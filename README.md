# Fintech Dashboard

This project is a modern fintech dashboard built with Next.js, TypeScript, Tailwind CSS, Appwrite, Plaid, and Dwolla. It is designed to give users a simple and secure way to manage their finances from a single web application.

## What this project does

The dashboard allows users to:

- sign up and sign in securely
- connect multiple bank accounts
- view their total balance and recent transactions
- track spending by category
- browse transaction history and account details
- transfer money to other users

## Why it exists

The goal of this project is to make personal finance management easier and more transparent. Instead of switching between different tools, users can monitor balances, review account activity, and move money from one place.

## Main technologies

- Next.js
- TypeScript
- Tailwind CSS
- Appwrite
- Plaid
- Dwolla
- ShadCN UI

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.
